# Fashion & Beauty Supply Chain Dashboard
This Power BI project visualizes and analyzes the supply chain of a fashion and beauty company using real-time data. The dashboard helps monitor inventory, vendor performance, product movement, and optimize supply chain operations.

📊 Files Included
- `SupplyChainData.pbix` – Main Power BI report
- `supply_chain_data.csv` – Source dataset
- `Dashboard SCD.jpg` – Image preview of the dashboard

🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel / CSV
- Data Modeling & DAX

🔍 Insights Offered
- Vendor-wise performance
- Product-level tracking
- Inventory bottlenecks
- KPI monitoring

🚀 Future Improvements
- Real-time API-based data integration
- Predictive analytics for demand forecasting
